# Automação de Relatórios com Python

## Sobre o Projeto
Este projeto tem como objetivo automatizar o processo de coleta, armazenamento e análise de dados, gerando relatórios de forma automática.

A aplicação consome dados de uma API pública, armazena essas informações em um banco de dados relacional e executa consultas SQL para gerar relatórios, reduzindo a necessidade de tarefas manuais.

---

## Motivação
Projeto desenvolvido com foco em praticar e demonstrar habilidades exigidas no mercado de tecnologia, especialmente:

- Manipulação de dados com Python
- Integração com APIs
- Uso de banco de dados SQL
- Automação de processos
- Organização de código em módulos

---

## Funcionalidades

✔ Coleta de dados via API REST  
✔ Armazenamento em banco de dados (SQLite)  
✔ Execução de consultas SQL  
✔ Geração automática de relatório em arquivo `.txt`  
✔ Estrutura modular e organizada  

---

## Tecnologias Utilizadas

- Python 3  
- SQLite  
- Requests (consumo de API)  
- Git & GitHub
